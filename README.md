# Fotostock

   Приложение загружает случайные фотографии с бесплатного фотостока. Также имеет возможность поиска по ключевым словам и дабавление в избранное понравившихся фотографий

При разработке использовал:
* JSON
* CocoaPods
* ScrollView
* Cетевые запросы
* CocoaPods
